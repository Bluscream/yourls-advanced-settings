# Domain Settings Bridge for YOURLS

Allows defining YOURLS options (such as Turnstile site keys, Public Shortener titles/colors, and Logo Suite customizations) individually per requested domain name, with a fallback default profile.

## Installation

1. Copy this folder into your YOURLS `user/plugins` directory.
2. Activate the plugin in the YOURLS administration panel.
3. Configure your domain mappings under **Plugins -> Domain Profiles**.
